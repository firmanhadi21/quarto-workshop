Analisis Data dengan Quarto

Statistik deskriptif · plotting · regresi linear · geostatistik

🌐 English version: read this in English

Halaman ini menunjukkan analisis nyata — dijalankan langsung dari data — untuk R dan Python. Studi kasus: tingkat kemiskinan 35 kabupaten/kota di Jawa Tengah (data/data_kemiskinan_jateng.csv). Salin kodenya dan jalankan di komputer Anda.

library(dplyr); library(ggplot2); library(knitr)
df <- read.csv("data/data_kemiskinan_jateng.csv")
df23 <- df[df$tahun == 2023, ]

1. Statistik deskriptif

Ringkasan sebaran variabel: pemusatan (rata-rata, median) dan keragaman (simpangan baku, rentang).

  • Python

Visualisasi membantu memahami sebaran (histogram) dan hubungan (diagram pencar).

3. Regresi linear berganda

Memodelkan kemiskinan sebagai fungsi beberapa variabel bebas. Perhatikan tanda koefisien (arah pengaruh), p-value (signifikansi), dan R² (daya jelas model). Pakai data panel 2019–2023 (175 observasi).

  • R
  • Python
model <- lm(kemiskinan ~ pdrb_kapita + inflasi + ipm + pengangguran, data = df)
koef <- summary(model)$coefficients[, c("Estimate", "Pr(>|t|)")]
rownames(koef) <- c("Konstanta","PDRB per kapita","Inflasi","IPM","Pengangguran")
kable(koef, col.names = c("Koefisien", "p-value"), digits = 3)
Koefisien p-value
Konstanta 98.355 0.000
PDRB per kapita -0.123 0.001
Inflasi 0.198 0.112
IPM -1.077 0.000
Pengangguran 0.235 0.031 
cat(sprintf("R-squared = %.3f", summary(model)$r.squared))
R-squared = 0.533
import statsmodels.formula.api as smf
model = smf.ols("kemiskinan ~ pdrb_kapita + inflasi + ipm + pengangguran", data=df).fit()
koef = pd.DataFrame({"Koefisien": model.params, "p-value": model.pvalues}).round(3)
koef.index = ["Konstanta","PDRB per kapita","Inflasi","IPM","Pengangguran"]
print(koef)
                 Koefisien  p-value
Konstanta           98.355    0.000
PDRB per kapita     -0.123    0.001

IPM                 -1.077    0.000
Pengangguran         0.235    0.031
print(f"R-squared = {model.rsquared:.3f}")
R-squared = 0.533

Membaca hasil: tanda koefisien = arah pengaruh (mis. koefisien negatif pada IPM ⇒ makin tinggi IPM, makin rendah kemiskinan); p-value < 0,05 ⇒ pengaruh signifikan; R² = proporsi variasi yang dijelaskan model. R² yang rendah pada model sederhana ini justru wajar — data panel sering butuh efek tetap (fixed effects) atau regresi spasial (lihat buku). Itulah gunanya reproducible: ganti model, render ulang, bandingkan.

4. Geostatistik

Data spasial punya pesan tambahan: apakah daerah miskin mengelompok? Kita petakan sebarannya, lalu uji autokorelasi spasial dengan Moran’s I.

Peta tematik (choropleth)

  • R
  • Python
library(sf); library(maptiles); library(tidyterra)
geo <- st_transform(st_read("data/jateng.geojson", quiet = TRUE), 3857)
peta <- merge(geo, df23, by = "wilayah", all.x = TRUE)
latar <- get_tiles(peta, provider = "OpenStreetMap", zoom = 8, crop = TRUE)
ggplot() + geom_spatraster_rgb(data = latar) +
  geom_sf(data = peta, aes(fill = kemiskinan), color = "white", linewidth = 0.1, alpha = 0.85) +
  scale_fill_distiller(palette = "OrRd", direction = 1, name = "Kemiskinan (%)") + theme_void()
import geopandas as gpd, contextily as ctx
g = gpd.read_file("data/jateng.geojson")
pmap = g.merge(df23, on="wilayah", how="left").to_crs(3857)
ax = pmap.plot(column="kemiskinan", cmap="OrRd", legend=True, alpha=0.85,
               edgecolor="white", linewidth=0.3, figsize=(8, 4.2))
ctx.add_basemap(ax, source=ctx.providers.OpenStreetMap.Mapnik)
ax.set_axis_off(); plt.show()

Autokorelasi spasial — Moran’s I

Moran’s I positif & signifikan ⇒ daerah dengan nilai serupa berdekatan (mengelompok).

  • R
  • Python
library(spdep)
nb <- poly2nb(peta)                 # ketetanggaan (queen contiguity)
w  <- nb2listw(nb, style = "W")     # bobot spasial
mt <- moran.test(peta$kemiskinan, w)
cat(sprintf("Moran's I = %.3f (p = %.4f)", mt$estimate[1], mt$p.value))
Moran's I = 0.294 (p = 0.0038)
from libpysal.weights import Queen
from esda.moran import Moran
w = Queen.from_dataframe(pmap, use_index=False); w.transform = "r"
mi = Moran(pmap["kemiskinan"].values, w)
print(f"Moran's I = {mi.I:.3f} (p = {mi.p_sim:.4f})")
Moran's I = 0.294 (p = 0.0110)

Tafsir: bila Moran’s I positif & signifikan, kemiskinan mengelompok secara spasial — ada “kantong” kemiskinan. Lanjutkan dengan LISA (klaster lokal) dan regresi spasial bila perlu (lihat buku, bab Statistik Spasial).
